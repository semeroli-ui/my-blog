关于这个博客

一个使用Flask构建的个人技术博客，记录学习、分享知识

项目介绍

这是一个使用 Python Flask 框架构建的个人博客系统，具备完整的文章发布、展示和管理功能。

项目采用简洁的MVC架构，结合SQLite数据库、Jinja2模板引擎和Bootstrap前端框架，实现了响应式设计，在电脑和手机上都能良好展示。

技术栈
  • Python 3.10 后端语言
  • Flask 2.3.x Web框架
  • SQLite 数据库
  • Bootstrap 5 前端框架
  • Jinja2 模板引擎
功能特性
  • 文章发布与管理
  • 响应式设计，支持移动端
  • 文章详情页展示
  • 简洁美观的用户界面
  • 自动字数统计与阅读时间估算
  • SQLite数据库存储
  • Bootstrap 5现代化设计
